Description of this folder:

  * Case 3 (parametric analysis): Recent HGT reported by Lawrence and Ochman in E. coli.
            - CDS analyzed with multiple genomic indices (using codonw).
            - Figures in PDF format illustrating the intersection between CAI vs GC% or GC3s vs GC%.
            - R files (*.RData) sessions (saved to reproduced the results).
    

Example files:
  - INPUT.file: MY_CODONW_DATA_cds_from_genomic.out.txt
  - OUTPUT.file: RicCom.CDS_genome.pdf          (example 1)
  - RDATA.saved: RicCom.CDS_genome.RData        (example 1)
  
  - OUTPUT.file: Rdefault_output.figure.pdf     (example 2)
  - RDATA.saved: default_output.Rsession.RData  (example 2)
  
  - OUTPUT.file: myPLOT_example1.pdf            (example 3)


 Reference:
    * Case 3: J. Mol. Evol. 44: 383-397 (1994).
